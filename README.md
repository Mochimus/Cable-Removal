Description: Remove a all the cables from a patch  panel or a switch, or world.

To delete all the cables from a switch or patch panel just look at one and hold leftctrl + right click.
To delete all the cables from the world look away from a switch or patch panel and hold for 10 seocnds leftctrl + right click.
To change the keybinds load the game once with the .dll inside the mods folder. Than close the game and  in the mods folder there will be a MassCableRemover_Keybinds file created. Just open it and change make the changes there.
